# Woody Business 

This is a Next.js project for "Woody Business", an e-commerce store selling wooden products like keychains, wall hangings, and more.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/mysamargroup-boop/woody-business.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

## Usage

To start the development server, run:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Deployment

This project can be deployed to Vercel.

## Project Structure

*   `src/app/`: Contains the main application code.
*   `src/lib/products.json`: Contains the product data.
*   `public/`: Contains static assets like images.
*   `next.config.mjs`: Next.js configuration file.
