Please place your Excel file with product information in this directory.

Once you have added the file, you can ask the AI to read it and update the products in your store.

**Excel File Format:**

Your Excel file should have at least the following columns:

- `id`: The unique product ID (SKU) that matches an existing product in `src/lib/products.json`.
- `title`: The new product title.
- `description`: The new product description.
